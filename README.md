## 8. 串口中断

```c
//中断标志位设置，根据条件查表初值
//发送程序  接收程序

//串口通用接收程序
uchar receive()
{
	uchar dat;
	while(RI == 0);
	RI = 0;
	dat = SBUF;
	return dat;
}

//串口通用发送程序
void send(uchar dat)
{
	SBUF = dat;
	while(TI == 0);
	TI = 0;
}
```
